# تطبيق قائمة المشاهدة - Watchlist App

تطبيق React Native مع Expo لإدارة قائمة الأفلام والمسلسلات والأنمي مع ذكاء اصطناعي متكامل.

## المميزات

### ✨ المميزات الرئيسية
- 📱 تصميم iOS Native Stack Navigation
- 🎨 واجهة مستخدم مستوحاة من iOS
- 🌙 وضع ليلي/نهاري
- 🤖 ذكاء اصطناعي مدمج (Gemini API)
- 💾 تخزين محلي للبيانات
- 🎬 إدارة الأفلام والمسلسلات والأنمي

### 🎯 الأقسام الأربعة
1. **الرئيسية**: عرض سريع للمحتوى مع زر الذكاء الاصطناعي
2. **الأفلام**: إدارة قائمة الأفلام
3. **المسلسلات**: إدارة قائمة المسلسلات والأنمي
4. **الإعدادات**: تفعيل الوضع الليلي والمعلومات

### 🤖 الذكاء الاصطناعي
- واجهة محادثة بتصميم iOS
- دعم مفتاحين من Gemini API مع نظام Fallback تلقائي
- حفظ سجل المحادثات
- إنشاء محادثات جديدة

### 📝 إدارة المحتوى
- إضافة أفلام/مسلسلات (اسم، وصف، صورة)
- تحديد الحالة: (سأشاهده، أشاهده الآن، تم المشاهدة)
- عرض تفاصيل كاملة
- حذف العناصر

## التثبيت والتشغيل

### المتطلبات
- Node.js 18+
- npm أو yarn
- Expo CLI

### خطوات التثبيت

1. **تثبيت المكتبات**
```bash
cd WatchlistApp
npm install
```

2. **إضافة مفاتيح Gemini API**
افتح الملف `services/AIService.js` وضع مفاتيح API الخاصة بك:

```javascript
const GEMINI_API_KEYS = [
  'YOUR_FIRST_API_KEY_HERE',  // المفتاح الأول
  'YOUR_SECOND_API_KEY_HERE', // المفتاح الثاني
];
```

للحصول على مفتاح API:
- اذهب إلى: https://makersuite.google.com/app/apikey
- سجل دخول بحساب Google
- أنشئ مفتاح API جديد
- انسخ المفتاح

3. **تشغيل التطبيق**
```bash
npx expo start
```

سيفتح Expo Dev Tools، يمكنك اختيار:
- `i` - تشغيل على iOS Simulator
- `a` - تشغيل على Android Emulator
- مسح QR Code من تطبيق Expo Go على هاتفك

## كيفية الاستخدام

### إضافة فيلم أو مسلسل
1. اذهب لقسم الأفلام أو المسلسلات
2. اضغط زر + العائم
3. أدخل الاسم (مطلوب)
4. أدخل الوصف (اختياري)
5. أدخل رابط الصورة (اختياري)
6. اختر الحالة
7. اضغط حفظ

### استخدام الذكاء الاصطناعي
1. من الشاشة الرئيسية، اضغط على زر "الذكاء الاصطناعي"
2. اكتب سؤالك في حقل الإدخال
3. اضغط زر الإرسال
4. للوصول للمحادثات السابقة، اضغط زر القائمة في الأعلى

### تغيير الحالة
1. اضغط على أي فيلم/مسلسل لعرض التفاصيل
2. في قسم "الحالة"، اختر الحالة الجديدة
3. سيتم الحفظ تلقائياً

## التخصيص

### تغيير الألوان
عدّل ملف `theme.js` لتغيير الألوان:

```javascript
export const lightTheme = {
  background: '#F2F2F7',
  primary: '#007AFF',
  // ... باقي الألوان
};
```

### إضافة مفتاح API ثالث
في `services/AIService.js`:

```javascript
const GEMINI_API_KEYS = [
  'FIRST_KEY',
  'SECOND_KEY',
  'THIRD_KEY',  // أضف مفاتيح إضافية
];
```

## ملاحظات مهمة

1. **مفاتيح API**: تأكد من إضافة مفاتيح Gemini API الصحيحة
2. **الصور**: استخدم روابط صور صالحة (https://)
3. **اللغة**: التطبيق مصمم باللغة العربية بالكامل
4. **التصميم**: يحاكي iOS بشكل دقيق (خطوط، ألوان، انتقالات)

## المشاكل الشائعة وحلولها

### لا يعمل الذكاء الاصطناعي
- تأكد من إضافة مفاتيح API صحيحة
- تحقق من الاتصال بالإنترنت
- تأكد من أن المفاتيح لم تتجاوز الحد المسموح

### لا تظهر الصور
- تأكد من استخدام روابط HTTPS
- تحقق من صلاحية الرابط في المتصفح

### التطبيق بطيء
- أعد تشغيل التطبيق
- امسح cache في Expo: `npx expo start -c`
