# دليل استخدام الذكاء الاصطناعي 🤖

## نظرة عامة

يستخدم التطبيق **Google Gemini AI** لتوفير مساعد ذكي يمكنه:
- الإجابة على أسئلتك
- تقديم اقتراحات للأفلام والمسلسلات
- المساعدة في كتابة الأوصاف
- الدردشة العامة

## كيفية الحصول على مفتاح API مجاناً

### الطريقة 1: Google AI Studio (الأسهل)

1. **افتح الرابط**: https://makersuite.google.com/app/apikey

2. **سجل الدخول** بحساب Google الخاص بك

3. **أنشئ مفتاح جديد**:
   - اضغط "Create API Key"
   - اختر "Create API key in new project" أو اختر مشروع موجود
   - انتظر ثوانٍ قليلة

4. **انسخ المفتاح**:
   - سيظهر المفتاح بصيغة: `AIzaSy...`
   - اضغط على أيقونة النسخ
   - احتفظ به في مكان آمن

### الطريقة 2: Google Cloud Console (متقدمة)

1. اذهب إلى: https://console.cloud.google.com
2. أنشئ مشروع جديد أو اختر موجود
3. فعّل "Generative Language API"
4. انتقل إلى "Credentials"
5. أنشئ "API Key"

## إضافة المفتاح للتطبيق

### الخطوة 1: افتح الملف
```
services/AIService.js
```

### الخطوة 2: ابحث عن هذا الكود
```javascript
const GEMINI_API_KEYS = [
  'AIzaSyDPUT8YourFirstAPIKeyHere',  // المفتاح الأول
  'AIzaSyDPUT8YourSecondAPIKeyHere', // المفتاح الثاني
];
```

### الخطوة 3: استبدل بمفاتيحك
```javascript
const GEMINI_API_KEYS = [
  'AIzaSyAbc123456789YourActualKey',     // ضع مفتاحك الحقيقي هنا
  'AIzaSyDef987654321YourSecondKey',     // مفتاح احتياطي (اختياري)
];
```

### الخطوة 4: احفظ الملف
اضغط Ctrl+S (Windows/Linux) أو Cmd+S (Mac)

## كيف يعمل نظام المفتاحين؟

التطبيق يستخدم نظام **Fallback ذكي**:

```
1. يحاول استخدام المفتاح الأول
   ✅ نجح؟ → يستمر في استخدامه
   ❌ فشل؟ → ينتقل للمفتاح الثاني

2. إذا فشل المفتاح الثاني أيضاً
   → يعرض رسالة خطأ للمستخدم
```

### أسباب فشل المفتاح:
- 🚫 نفد الحد المجاني (60 طلب/دقيقة)
- 🚫 المفتاح غير صحيح
- 🚫 مشكلة في الاتصال بالإنترنت
- 🚫 المفتاح معطل أو منتهي

## الحدود المجانية لـ Gemini

### حد الاستخدام المجاني:
- ✅ **60 طلب في الدقيقة**
- ✅ **1500 طلب في اليوم**
- ✅ غير محدود شهرياً

### نصائح لتجنب تجاوز الحد:
1. لا ترسل طلبات كثيرة جداً بسرعة
2. استخدم مفتاحين مختلفين (حسابين Google)
3. إذا نفد الحد، انتظر دقيقة وجرب مرة أخرى

```

```

### مثال 2: كتابة وصف


## حل المشاكل الشائعة

### ❌ خطأ: "جميع مفاتيح API غير متاحة حالياً"

**السبب المحتمل:**
- المفاتيح غير صحيحة
- تجاوزت الحد المسموح

**الحل:**
1. تحقق من المفاتيح في `services/AIService.js`
2. انتظر دقيقة وحاول مرة أخرى
3. أنشئ مفتاح جديد

### ❌ خطأ: "لم يتم استلام رد صحيح من الخادم"

**السبب:**
- مشكلة في الاتصال بالإنترنت
- Gemini API معطل مؤقتاً

**الحل:**
1. تحقق من اتصالك بالإنترنت
2. حاول مرة أخرى بعد قليل

### ❌ الرسائل بطيئة جداً

**السبب:**
- اتصال إنترنت ضعيف
- خادم Gemini مشغول

**الحل:**
- انتظر قليلاً، الذكاء الاصطناعي يعالج السؤال

## الأمان والخصوصية

### ⚠️ معلومات مهمة:
- 🔒 لا تشارك مفتاح API مع أحد
- 🔒 لا تنشر المفتاح على GitHub
- 🔒 المحادثات محفوظة محلياً فقط على جهازك
- 🔒 Gemini قد يحفظ المحادثات لتحسين الخدمة

### إضافة المفتاح إلى .gitignore (للمطورين):

إذا كنت ستنشر الكود على GitHub، أنشئ ملف `.env`:

```bash
# .env
GEMINI_API_KEY_1=AIzaSy...
GEMINI_API_KEY_2=AIzaSy...
```

ثم عدّل `AIService.js`:
```javascript
import Constants from 'expo-constants';

const GEMINI_API_KEYS = [
  Constants.expoConfig.extra.GEMINI_API_KEY_1,
  Constants.expoConfig.extra.GEMINI_API_KEY_2,
];
```
